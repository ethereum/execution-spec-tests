Running Tests

This section documents how EL clients can run EEST tests to verify their implementations. The information is organized as follows:

  1. Methods of Running Tests: A detailed overview and comparison.

  2. EEST Fixture Releases: The available release types; release versioning.

  3. Test Fixture Specifications and guides to implement direct consumer interfaces.

  4. Hive: Help running EEST Hive simulators.

  5. Consume: Commands that help clients run tests by consuming test fixtures.

  6. Execute: Commands that help clients run tests by executing test source.
