Consume Simulators

The engine and rlp simulators test clients by importing blocks through different interfaces. These simulators run within the Hive testing framework to provide containerized, isolated testing environments.

Command Syntax

uv run consume <engine|rlp> [OPTIONS]

Relevant Information

  • To install the consume command, see Installation.
  • Help setting up and starting Hive in dev mode.
  • For an explanation of how the consume simulators work, see the Engine and RLP sections in Running Tests.
  • Help for relevant options can be found in Consume Cache and Fixture Inputs and Useful Pytest Options.
